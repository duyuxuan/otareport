# 1.重要说明
otareport是一个超轻量级的报表平台。前端使用的html5框架amazeui+fushingcharts，后端是jfinal-weixin。
欢迎大家拍砖，如果有商务合作，欢迎联系。QQ：857169072，留言请填写github。

# 2.使用方法：
download项目，配置jkd1.8，tomcat8、9，创建数据库，导入doc/sql.sql，运行即可。
# 3、目录说明：
com.base基础帮助类
com.data报表平台的api
com.jfinal是jfinal框架的内容，内有做的扩展
com.webao暂未使用

webapp/report 报表页面
